
# Well Computer - Smart Sales System

Aplikasi Manajemen Penjualan Laptop berbasis Web yang aman dan scalable.

## Struktur Folder (WAJIB SEPERTI INI)

Pastikan saat upload ke GitHub, strukturnya tetap begini:
```text
/ (root)
├── index.html
├── App.tsx
├── index.tsx
├── types.ts
├── constants.tsx
├── vite.config.ts
├── package.json
├── components/
│   ├── Layout.tsx
│   ├── Dashboard.tsx
│   ├── Inventory.tsx
│   ├── WhatsAppHub.tsx
│   ├── TransactionForm.tsx
│   ├── TransactionsList.tsx
│   ├── Reports.tsx
│   ├── UserManagement.tsx
│   ├── StoreManagement.tsx
│   ├── BackupManager.tsx
│   ├── Login.tsx
│   └── AIAssistant.tsx
└── services/
    └── geminiService.ts
```

## Tips Upload dari HP
Jika menggunakan browser, buatlah folder `components` dan `services` terlebih dahulu di GitHub dengan cara klik **"Create new file"** lalu ketik nama folder diikuti garis miring (contoh: `components/`).
